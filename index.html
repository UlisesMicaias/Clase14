Cuando realizamos un request, lo realizamos usando un metodo o method. Estos pueden ser:

  • GET: Socitar información
  • POST: Enviar información//crear
  • PUT: Editar
  • DELETE: Eliminación

Dentro del response, viene incluido entre otras opciones un statusCode o codigo de estado, referido a la operación que estemos realizando. Estos suelen ser:

  • 1xx: Código de información
  • 2xx: Operación exitosa
  • 3xx: Redirecciona para completar las operaciones mediante una nueva URL
  • 4xx: Error por parte del cliente
  • 5xx: Error por parte del servidor
